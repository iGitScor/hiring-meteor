# Hiring meteor

Building Angular2 UI for a Meteor app : recruitment website.

## Installing

```shell
meteor --port 3000
```

## Developing

```shell
git clone https://github.com/{your fork}/hiring-meteor.git
cd hiring-meteor/
meteor --port 3000
open localhost:3000
```

Replace `{your fork}` by your github username.
